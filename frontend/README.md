# Task Management System - Frontend

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Features Implemented

### Week 4
- **Task Sharing**: Share tasks with other users via the frontend Share button.
- **Frontend Error Handling**: Clear error messages displayed for failed actions.
- **Notifications**: Real-time notifications integrated into the UI.

### Week 5
- **Analytics Dashboard**:
  - Overview of tasks (total, completed, pending, in-progress).
  - Trends view (weekly/monthly completed vs overdue tasks).
- **Data Visualization**: Charts integrated using Chart.js/Recharts.

### Week 6
- **File Attachments**:
  - Upload files directly to tasks.
  - Attachments displayed in Task Details with clickable download links.
- **Dark Mode Toggle**: Switch between light and dark themes, with preference saved.
- **Mobile Responsiveness**: Layouts optimized for smaller screens.
- **Documentation & Testing**: Enhanced README, screenshots, and end-to-end testing.
